Classlist

La propiedad classlist devuelve los nombres de clase de un elemento. Esta propiedad es util para agregar, eliminar y alternar clases CSS en un elemento.

La propiedas classlist es de solo lectura, sin embargo, puede modificarse utilizando estos metodos:

  • add(): La propiedad classlistmas el metodo add, agrega uno o mas nombres de clase a un elemento.
  • remove(): La propiedad classlist mas el metodo remove elimina o uno o mas nombres de clase de un elemento
  • toggle(): La propiedad classlist mas el metodo toggle alterna entre un nombre de clase para un elemento.

Ideas claves

  • La propiedad classlist no es soportada en IE9 y versiones anteriores.
  • En classlist.toggle el segundo parametro no es compatible con internet explore, opera 12 y versiones anteriores.



Resumen

Eventos

En HTML DOM le permite ejecutar codigo cuando ocurre un evento.
El navegador genera eventos cuando "suceden cosas" a elementos HTML como por ejemplo:

  • Al hacer click con el mouse "onclick"
  • Al hacer doble click con el mouse "ondblclick"
  • Al colocar el mouse sobre un elemento "onmouseover"
  • Al alejar el mouse de un elemento "onmouseout"
  • Cuando una pagina web se ha cargado "onload"
  • Al presionar una tecla "onkeydown"
  • Al hacer scroll en la pagina "onscroll"

El metodo addventlistener() adjunta un controlador de eventos al elemento especificado. La sintaxis para este metodo es la siguiente: elemento.addventlistener(evento, funcion..)

If else

Una declaracion condicional se usa para determinar el flujo de programa basado en una condicion.

En Javascript tenemos las siguientes declaraciones condicionales:

  • If: Si es verdadera, se ejecuta el bloque de codigo correspondiente y se salta el resto de las condiciones.
  • Else: Este bloque se ejecuta si ninguna de las condiciones anteriores fue verdadera. Es opcional, pero util para manejar casos no previstos.
  • Else if: Si es verdadera se ejecuta su bloque de codigo y se termina la evaluacion. Si es falsa se pasa a la siguiente condicion.

La sintaxis es: if (condicion) {
}
else {
}

Las declaraciones else if en Javascript permiten evaluar multiples condiciones secuencialmente. Esto es util cuando se tienen mas de dos posibles situaciones que manejar en el codigo.

  • Se utilizan los operadores de comparacion.
  • En el if se ejecuta si es verdadera y si es falsa pasa a la siguiente condicion.
  • El else if permite evualar multiples condiciones.

@ Media

Estos son un modulo de CSS3 que adapta la representacion de contenido a las caracteristicas de un dispositivo. Se convirtio en un estandar recomendado por la W3C en junio de 2012 y es un principio basico del dise√±o web adaptativo.

Se utilizan para verificar varias cosas como: ancho y alto de la ventana grafica, ancho y alto del dispositivo, orientacion del dispositivo y la resolucion.

El uso de media queries es una tecnica popular para entregar una hoja de estilo personalizada para:

  • Computadoras de escritorio
  • Computadoras portatiles
  • Tabletas
  • Telefonos moviles (como los telefonos Iphone y Android)

Tipos de media

  • All: Se utiliza para todos los dispositivos de tipo de medios.
  • Print: Utilizado para impresoras.
  • Screen: Se utiliza para pantallas de ordenador, tabletas, telefonos inteligentes, etc.
  • Speech: Se utiliza para lectores de pantalla que "lee" la pagina en voz alta.

Ideas claves

  • La idea de media queries fue de hacer que en lugar de buscar un tipo de dispositivo se analiza la capacidad del dispositivo.
